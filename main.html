Bienvenido a SUPPRIME

HTML
Es un lenguaje de etiquetas que permite la creación de paginas y sitios web y es una de las tecnologías fundamentales e imprescindible para la creacion de una pagina
¡Ideal para empezar!
CSS
Se le conoce como las hojas de estilo en cascada permiten mejorar el aspecto visual de paginas y sitios web, ademas de ser una de las tecnologías fundamentales del desarollo web en aplicaciones de todo tamaño.
Javascript
Es el lenguaje de la web. Es un lenguaje de scripting que puedes usarlo en cualquier entorno desde aplicaciones web, móviles, aplicaciones de servidor e incluso aplicaciones de escritorio.
SQL
es un acrónimo en inglés para Structured Query Language. Un Lenguaje de Consulta Estructurado. Un tipo de lenguaje de programación que te permite manipular y descargar datos de una base de datos.
React
Es el lenguaje de la web. Es un lenguaje de scripting que puedes usarlo en cualquier entorno desde aplicaciones web, móviles, aplicaciones de servidor e incluso aplicaciones de escritorio.
Node js
Es un entorno de ejecucion que permite a Javascript ejecutarse fuera del navegador permitiendo crear aplicaciones web backend, aplicaciones de Escritorio, aplicaciones Móviles e incluso Robots
MongoDB
Es una base de datos NoSQL, que utilizar a Javascript como lenguaje de consultas y a JSON como formato para poder guardar datos. Esta pensada para escalar en proyectos grandes y posee flexibilidad a diferencia de SQL
Express
Es un ramework Backend de Nodejs, una infraestructura de aplicaciones web Node.js mínima y flexible que proporciona un conjunto sólido de características para las aplicaciones web y móviles.
